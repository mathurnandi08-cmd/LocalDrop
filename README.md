# LocalDrop v1.0 📡

LocalDrop is a professional, wireless file transfer system built with Python and Flask. It allows users to transfer files from a mobile device to a PC seamlessly over a local network using a QR code.

## ✨ Features
- **Zero Configuration:** Automatically detects your Local IP.
- **QR Code Pairing:** Scan and connect instantly—no cables required.
- **Glassmorphism UI:** A beautiful, modern, and responsive mobile upload page.
- **Secure Storage:** Files are saved directly to a dedicated folder on your `D:` drive.
